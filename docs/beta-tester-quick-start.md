# Leftly Beta Tester Quick Start

Use this guide if you want a simple way to try Leftly without guessing what matters.

## How to test Leftly

You can use sample or fake numbers if you do not want to use real paycheck details yet.

### Basic checklist

- Start a new paycheck setup.
- Enter the paycheck amount and dates.
- Add 2-3 bills.
- Leave setup and come back to confirm the setup draft stays saved.
- Complete setup.
- Add a manual expense.
- Add a one-time bill.
- Check Bill Plan.
- Check History.
- Export a JSON backup.
- Export CSV if you want spreadsheet records.
- Try browser install or add to home screen if your browser supports it.

## What not to worry about

- Leftly does not connect to bank accounts.
- Leftly does not require an account for local use on your device.
- Browser install or add to home screen is optional.
